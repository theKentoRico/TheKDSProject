# Keywords

1. Errorc (KDS 1.0.1)
Errorc makes the program return the exit code written after it.

### Usage

> errorc *x*;

where x is an integer

2. Value (KDS 2.0.3)
### Usage
> value x \{*valtype*\} = *y*;

where valtype is a variable type like long ( i64 ), int ( i32 ), char ( i8 ) or bool ( i1 ).
